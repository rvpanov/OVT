# Страница статуса (Status Page)

Автономная HTML-страница, которая отображает и переключает статус (ON/OFF) исключительно на стороне клиента (в браузере). Не требует запуска Python-сервера, установки дополнительных программ или подключения к интернету.

## Описание
Файл `status.html` содержит встроенный JavaScript-скрипт, который анализирует адресную строку браузера. Если в URL присутствует специальный параметр (`?status=on` или `?status=off`), страница мгновенно обновляет отображаемый текст, не требуя перезагрузки. По умолчанию, если параметр не указан, отображается статус `OFF`.

## Требования
* Любой современный веб-браузер (Google Chrome, Mozilla Firefox, Safari, Microsoft Edge и др.)
* **Не требуется:** Python, Flask, Node.js или любой другой сервер.

## Использование
1. Сохраните файл `status.html` на вашем компьютере.
2. Откройте файл двойным щелчком мыши (или перетащите его в окно открытого браузера).
3. По умолчанию вы увидите надпись: **Status: OFF**.
4. Чтобы изменить статус, добавьте в самый конец адресной строки браузера соответствующий параметр и нажмите `Enter`:
   - **Чтобы включить:** добавьте `?status=on`  
     *(Пример: `file:///C:/project/status.html?status=on`)*
   - **Чтобы выключить:** добавьте `?status=off`  
     *(Пример: `file:///C:/project/status.html?status=off`)*

## Как это работает
При загрузке страницы или изменении URL скрипт выполняет следующие действия:
1. Считывает параметры запроса из адресной строки (`window.location.search`).
2. Ищет параметр с именем `status`.
3. Если найдено значение `on` или `off` (регистр не важен), обновляет текст на странице.
4. Если параметр отсутствует или некорректен, по умолчанию устанавливает `OFF`.

## Ограничения
* **Отсутствие сохранения:** Статус не записывается в файл (например, в `status_log.txt`) и не сохраняется между разными браузерами или устройствами.
* **Зависимость от URL:** Если вы просто откроете файл без параметров в адресной строке, он всегда будет показывать `OFF`. Статус "живет" только пока в адресе есть нужный параметр.
