# Status Server

Простой веб-сервер на Python (Flask), который позволяет переключать статус (ON/OFF) через адресную строку браузера. Каждое изменение статуса сохраняется в файл с отметкой времени.

## Описание
Сервер запускается на локальном компьютере. Открыв браузер и вписав нужный адрес, можно:
- Включить статус (`/on`)
- Выключить статус (`/off`)
- Посмотреть текущий статус на весь экран (`/status`)
- Получить статус в виде текста для внешних программ (`/api/status`)

Все события записываются в файл `status_log.txt` и накапливаются — данные не перезаписываются при перезапуске сервера.

## Установка и запуск
Все действия выполняются в командной строке (терминале) по очереди:

1. Убедитесь, что Python установлен:

   python --version
   
2.Установите библиотеку Flask:
   
   pip install flask

3. Запустите сервер:

   python server.py

После запуска сервер будет доступен по адресу: http://localhost:5000
## Команды (маршруты)

| Адрес | Описание |
|---|---|
| `http://localhost:5000/` | Главная страница с подсказкой |
| `http://localhost:5000/on` | Записать статус ON с текущим временем |
| `http://localhost:5000/off` | Записать статус OFF с текущим временем |
| `http://localhost:5000/status` | Отобразить текущий статус на весь экран |
| `http://localhost:5000/api/status` | Получить статус в виде чистого текста (`on` или `off`) |

## Файл логов
Все события сохраняются в `status_log.txt` рядом с `server.py`. Файл создаётся автоматически при первом обращении к `/on` или `/off`.

**Пример содержимого:**

2026-06-14 14:22:01 - on

2026-06-14 14:45:33 - off

2026-06-14 15:10:07 - on
